LOGO PLACEMENT
==============

Place your "Jordie Block" script logo file here.

Accepted file name:  jordie-block-logo.png  (or .svg)
Recommended size:    400 × 120 px (transparent background, PNG or SVG)

To use it on the site, edit index.html and replace the nav-logo text link:

  CURRENT:
    <a href="#hero" class="nav-logo">Jordie Block</a>

  WITH IMAGE:
    <a href="#hero" class="nav-logo">
      <img src="assets/logo/jordie-block-logo.png" alt="Jordie Block" style="height:38px;width:auto;" />
    </a>

Do the same for .footer-logo in the footer section.
